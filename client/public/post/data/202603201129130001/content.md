# Java Script의 변수 예약어 특징 정리
1. var: ES6 전 Java Script의 유일한 변수 예약어. 해당 예약어는 매우 괴랄하고 사용하지 않을 것을 이곳 저곳에서 권고 중.
2. let: 대중적인 언어에서 사용하는 auto 혹은 var과 유사함.
3. const: 상수의 이름을 가졌으나 JS의 특징에 따라 상수라고만은 할 수 없음. 자바의 final이나 코틀린의 val과 사용법이 같음.

# 3. JS의 안전한 변수 선언 const
TS가 나오면서 readonly라는 키워드가 생겼다. 이유가 뭘까? JS엔 진정한 의미의 상수 저장이 없기 때문이다.
물론 const primitive의 경우 상수가 맞다. 일반적인 js 변수와 다르게 식별자 공간의 참조 값이 아니라 실제 값으로 변한다.
