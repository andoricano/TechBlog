# Tailwind와 Bootstrap 비교 테스트

## 목표
- CSS Framework 숙달
- Tailwind와 Bootstrap에 대한 차이 인식

## 중점적으로 볼 사안
- Tailwind : Component를 조합하는 방식으로 진행하여 CSS 코드를 구성하지 않아도 되는지 확인한다.
- Bootstrap : 완성품을 가지곤 오나 다시 CSS코드로 세밀 조정하는 부분을 살펴본다.
- 위 프레임워크를 일반적인 CSS 파일에 import할 경우 성능 문제가 생기는지 확인한다.

## 테스트 방법
- 포트폴리오 용 웹페이지를 제작
- 같은 페이지를 다른 프레임워크로 테스트 
- 2종류의 페이지를 중점적으로 생성
    1. 메인 페이지
    2. 성능 테스트 페이지
